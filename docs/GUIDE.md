# Meme Event — руководство организатора

Приложение для проведения турниров по **Warhammer 40,000** (сольных и
командных): автоматические парринги по швейцарской системе, подсчёт мест,
сдача результатов игроками с телефонов, рейтинг с историей и графиками.

Правила паррингов и подсчёта основаны на **Warhammer Event Companion** и
**Teams Event Companion** от Games Workshop, с поправками под наши турниры.

## Из чего состоит приложение

Три части, связанные через облако (Firebase):

| Часть | Где работает | Для кого |
|---|---|---|
| **Приложение организатора** | Windows / Android | ты ведёшь турнир |
| **Сайт игроков** — `meme-event.web.app` | браузер телефона | игроки сдают счёт и смотрят таблицу |
| **Сайт рейтинга** — `meme-event-rating.web.app` | любой браузер | все смотрят рейтинг, тиры, историю |

Приложение организатора — главное. Сайты только показывают то, что ты
**опубликовал**, и принимают сдачи результатов от игроков.

## Что нужно для старта

- **Компьютер на Windows** (или телефон на Android) для приложения организатора.
- **Интернет** — нужен, только чтобы игроки сдавали счёт онлайн и для рейтинга.
  Сам турнир можно вести полностью офлайн, а результаты вносить вручную.
- Игрокам — **любой телефон с браузером** (вводят код турнира или сканируют QR).

Турнир хранится локально на твоём устройстве, поэтому приложение можно
закрывать и открывать без потери данных.

## Словарь очков и терминов

Эти сокращения встречаются в таблицах и настройках:

- **ТО** — турнирные очки за игру: **2** победа / **1** ничья / **0** поражение.
- **VP** — очки за игру по шкале **0–20** (по таблице разницы счёта из Teams
  Companion). Сумма VP обоих игроков всегда равна 20. У нас это основной
  показатель «насколько уверенно выиграл».
- **GW** — «сырые» победные очки за партию, **0–100** (primary + secondary + покрас).
- **WPath** — «прогрессивка»: сумма ТО нарастающим итогом по турам. Ранние
  победы ценятся выше (WWL стоит больше, чем LWW).
- **SoS** — сила оппонентов (Strength of Schedule): сумма ТО всех твоих
  соперников. Чем сильнее соперники — тем выше.
- **Проксибот** — реальный игрок «вне зачёта», который проходит турнир
  бесплатно. Выходит на стол, только когда зачётных игроков нечётное число,
  и всегда играет с последним местом таблицы. Его результат предзаполняется
  «около ничьи» (настраивается).
- **Неявка** — техническое поражение: неявившемуся 0, явившемуся победа с
  настраиваемым счётом.
- **BP** (в командах) — те же VP (0–20), но суммируются по всей команде.
- **TP** (в командах) — командные очки за матч: **2** победа / **1** ничья /
  **0** поражение.
- **Тиры** (на сайте рейтинга): Diamond 1900+, Gold 1800+, Silver 1600+,
  Bronze 1500+, Iron ниже.

## Создание турнира

На главном экране нажми **«Новый турнир»** (кнопка внизу справа) и заполни:

1. **Название** — как будет называться турнир.
2. **Тип** — **Сольный** или **Командный**.
3. **Игроков в команде** (только для командного) — 3–8, по умолчанию **4**.
4. **Туров** — 3–6.
5. **Парринг первого тура:**
   - **По рейтингу** — верхняя половина против нижней (при 10 игроках:
     1-й с 6-м, 2-й с 7-м и т.д.). Для команд считается **сумма рейтингов**
     игроков команды.
   - **Случайно** — полный рандом (рекомендация GW).
   - **Вручную**  — пары первого тура расставляешь сам.
6. **Парринг туров 2+** — по каким метрикам сводить игроков
   в пары. Галочками включаешь нужные, перетаскиванием задаёшь приоритет.
   Внутри равных значений — случайно. По умолчанию: ТО → WPath.
   *(Команды туров 2+ всегда парятся по TP → индивидуальным победам → случайно,
   без рематчей.)*
7. **Тайбрейки** — чем разбивать равенство очков при дележе мест. Тот же
   набор метрик: галочки + перетаскивание приоритета. По умолчанию: ТО → VP → GW.
8. **Дополнительно** (свернутый блок) — счёт игры с проксиботом по умолчанию и
   сколько VP даётся явившемуся при неявке оппонента (по умолчанию 51 — это
   полная победа 20:0 по VP).

Нажми **«Создать турнир»** — откроется экран турнира с тремя вкладками:
**Игроки**, **Туры**, **Таблица**.

## Игроки и команды

Вкладка **«Игроки»**.

### Добавить игроков

- **Игрок** — вручную: имя и (необязательно) рейтинг.
- **Импорт из рейтинга** — подтягивает игроков из рейтинговой Google-таблицы:
  нажми, в диалоге уже подставлена ссылка, жми **«Загрузить»**, найди нужных
  через поиск и отметь галочками. Импортируются имя, рейтинг, RD, sigma.
  *(Неактивные игроки из таблицы не подтягиваются.)*
- **Проксибот** (только сольные) — добавляет бота-«вне зачёта» одной кнопкой.
  Нужен, если зачётных игроков нечётное число.

### Команды (командный турнир)

- **Команда** — создать команду (название).
- **Перетаскивай игроков мышкой** на карточку команды, чтобы распределить
  состав (или через меню игрока «Переместить в команду»). У каждой команды
  виден счётчик состава (например, «3 из 4»).
- Незанятые игроки лежат в блоке **«Без команды»**.

### Управление игроком (меню «⋮» справа)

- **Снять с турнира / Вернуть** — снятый игрок не участвует в паррингах
  следующих туров, но его сыгранные результаты остаются.
- **Сделать проксиботом / зачётным** (сольные).
- **Удалить** — доступно, пока игрок ещё не играл.

> Совет: добавь всех игроков **до** генерации первого тура. После старта
> состав менять можно (снять/вернуть), но это влияет на парринги.

## Проведение турнира по турам

Вкладка **«Туры»**.

### Генерация тура

- Кнопка **«Сгенерировать тур N»** внизу. Программа сама составит пары по
  выбранным правилам, без повторных встреч (рематчей).
- Кнопка **заблокирована**, пока не подтверждены все результаты текущего тура.
- При нечётном числе игроков последний в таблице играет с **проксиботом**
  (если он добавлен) или получает **bye** (пропуск тура с очками).

**Ручной первый тур** (если выбрал «Вручную»): тур 1 откроется пустым —
жми **«Добавить пару»** и расставляй вручную; внизу подсказка, кто ещё не
расставлен.

### Внесение результата

- Кликни по паре → введи **VP (GW)** каждого игрока (сырой счёт партии, 0–100) →
  **Сохранить**. Программа сама посчитает ТО и VP-очки (0–20).
- Кнопки **«Неявка: …»** — техническое поражение неявившемуся.
- Игра с проксиботом приходит с предзаполненным счётом «около ничьи» — поправь
  на фактический и подтверди.

### Статусы и прогресс

- В шапке каждого тура — счётчик **«сделано / всего»** (например, 5/8).
- Чип у пары: «нет счёта» → «ждёт подтверждения» → подтверждённый счёт.
- Меню тура (на последнем туре): **перегенерировать пары** или **удалить тур**.

### Командный матч

Каждый матч команда-на-команду показывает суммарный счёт **BP**. Внутри матча
пары назначаешь ты (это итог процедуры паррингов капитанов за столом):
**«Добавить пару»** → выбираешь по игроку из каждой команды. Тур закроется,
когда в каждом матче назначены все пары и внесены результаты.

## Таблица и места

Вкладка **«Таблица»** — обновляется после каждого подтверждённого результата.

- В командном турнире сверху **таблица команд** (TP, индивидуальные победы, BP),
  ниже **личный зачёт**.
- Личный зачёт: колонки **ТО / VP / GW / WPath / SoS** и счёт **В-Н-П**.
- Топ-3 отмечены медалями 🥇🥈🥉.
- Места считаются по выбранным **тайбрейкам** в заданном тобой порядке.
- Если есть неподтверждённые результаты — сверху появится предупреждение
  (такие результаты в таблицу пока не входят).

## Облако: публикация и сдача результатов игроками

Чтобы игроки видели турнир и сдавали счёт со своих телефонов:

1. **Опубликуй турнир** — кнопка-облачко в шапке экрана турнира. На ней горит
   точка, если есть неопубликованные изменения. Нажатие заливает турнир в
   облако. *Ничего не публикуется автоматически — только по этой кнопке.*
2. **Дай игрокам код** — он показан в шапке (например, «Код: K7N2PQ»),
   копируется кликом. Или нажми **кнопку QR** — покажи большой QR на экране/
   проекторе, игроки сканируют и попадают прямо в турнир.
3. **Игроки сдают счёт** со своего телефона. Сдача прилетает к тебе **сразу**:
   у пары появляется чип «50:40 — подтвердить».
4. **Подтверди** — кликни по паре и сохрани. Твоё подтверждение окончательно:
   игрок уже не сможет перезаписать подтверждённый счёт.

> После каждого тура нажимай «Опубликовать» ещё раз, чтобы у игроков обновились
> пары и таблица.

## Сайт игроков (что сказать участникам)

Дай участникам адрес **`meme-event.web.app`** и код турнира (или QR).

Что делает игрок:

1. Открывает сайт, **вводит код** (или сканирует QR — тогда код подставится сам).
2. **Выбирает себя** из списка (или заходит как «зритель»).
3. Видит карточку **«Твоя игра»**: тур, стол, соперник.
4. Жмёт **«Сдать счёт»**, вводит свои VP и VP соперника → отправляет.
   Результат попадает к организатору на подтверждение.
5. Во вкладках **«Туры»** и **«Таблица»** видит ход турнира (обновляется
   после твоих публикаций).

## Рейтинг

Рейтинг считается по системе **Glicko-2** (как в нашей мастер-таблице).

### Пересчёт после турнира

На вкладке **«Таблица»** → кнопка **«Пересчитать рейтинг (Glicko-2)»**.

- Откроется таблица **было → стало** с дельтой и новым RD. Новички стартуют с
  1500 / RD 153. В рейтинг идут только живые подтверждённые игры — проксибот,
  bye и неявки **не учитываются**.
- Рейтинг считается **за турнир одним блоком** (все игры разом), а не по партиям.
- Кнопка **«Скопировать для Google-таблицы»** кладёт результат в буфер
  (Игрок / Рейтинг / RD / Sigma) — вставляешь в свою таблицу через Ctrl+V.

Пересчёт ничего не меняет сам — это расчёт по текущим результатам, жать можно
сколько угодно.

### Сайт рейтинга

Адрес **`meme-event-rating.web.app`** — общий, для всех.

- **Поиск** игрока по имени, **фильтр по тирам**.
- Строки окрашены **по тиру** (Diamond / Gold / Silver / Bronze / Iron).
- Видно **движение** с прошлого турнира: смена места (↑/↓) и изменение рейтинга.
- **Тап по игроку** → график рейтинга по всем турнирам + сводка (текущий,
  пик, число турниров, изменение за всё время).

### Обновление сайта рейтинга

Чтобы на сайте появились свежие цифры и графики, на **главном экране**
приложения (список «Турниры») нажми кнопку с иконкой диаграммы —
**«Синхронизировать рейтинг на сайт»**. Она читает мастер-таблицу рейтинга
(все турниры) и заливает историю на сайт. Делай это после каждого обновления
рейтинга.

## Архив турниров

Завершённые турниры можно убрать с глаз, не удаляя:

- В меню карточки турнира (на главном экране) — **«В архив»**.
- Сверху появится переключатель **Активные / Архив**. В архиве — **«Вернуть
  из архива»**.
- **Удалить** стирает турнир насовсем (и из облака тоже).

## Частые вопросы и советы

**Нечётное число игроков.** Добавь проксибота (кнопка на вкладке «Игроки») —
он будет играть с последним местом. Без него последний получит bye (пропуск
с очками).

**Игрок опоздал/не пришёл.** Внеси его игру как **неявку** (кнопка в окне ввода
счёта). Если договорились о переносе — впиши счёт вручную.

**Ошибся в паррингах тура.** На последнем туре в меню тура есть
**«Перегенерировать пары»** или **«Удалить тур»**.

**Нет интернета на площадке.** Веди турнир как обычно, вноси счёт сам. Облако
и сдачу с телефонов подключишь, когда появится сеть (кнопка «Опубликовать»).

**Команды.** Нужно чётное число команд и одинаковый состав в каждой —
приложение не даст сгенерировать тур, пока составы не совпадают, и подскажет,
где не хватает игроков.

**Два компьютера / смена устройства.** Турнир привязан к устройству, с которого
его опубликовали — публиковать и обновлять его может только оно. Веди один
турнир с одного устройства.

**Имена игроков в рейтинге двоятся.** Рейтинг матчит игроков по имени, поэтому
пиши имена одинаково (одни и те же ники/кавычки), иначе история может
разделиться на двух «разных» людей.
